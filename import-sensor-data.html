Import sensor data

This documentation describes how to import sensor data in the SOS server instance using the feeder.

Requirements

  • SOS server is running
  • Configuration file has been created for the sensor station
  • Sensor data contains rows for a single station, no empty cells

Run from command line

Here is how you can run the feeder from the command line. Pass the configuration file with the -c parameter and the data file with the -d parameter. The console output is piped into a log file.

$ cd path/to/sos-importer
$ java -jar ./feeder/target/52n-sos-importer-feeder-bin.jar \
  -c ~/Desktop/SOS-Testdaten/sensordata.csv.52n-sos-import-config.xml \
  -d ~/Desktop/SOS-Testdaten/sensordata-010.csv \
  > ~/Desktop/sensordata-010.log
